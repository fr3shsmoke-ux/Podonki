# Telegram Bot — Модератор постов

## Как работает

```
Генератор идей (Этап 1) → 5 вариантов
    ↓
Бот показывает вариант 1-5
    ↓
Выбираешь: ✅ Принять / ✏️ Редактировать / ➡️ Далее
    ↓
Если редактируешь → отправляешь новый текст в боте
    ↓
Бот анализирует что изменил
    ↓
Пост попадает в очередь публикации (Этап 5)
```

## Команды бота

| Команда | Что делает |
|---------|-----------|
| `/start` | Главное меню |
| `/moderate` | Начать модерацию постов |
| `/stats` | Статистика модерации |

## Интерфейс

### Главное меню
```
🤖 Модератор постов Podonki

[📋 Модерировать посты]
[📊 Статистика]
[❓ Справка]
```

### Просмотр варианта
```
*Вариант 1/5*

📝 Теперь к стоматологу ходим только так...

Рубрика: мемы
Тон: шутливый
Score: 4.5

[✅ Принять]
[✏️ Редактировать]
[⬅️ Назад] [Далее ➡️]
[❌ Отказать всё]
```

### Комментарий (глобальное правило)
```
✅ Комментарий сохранен

📝 Вариант: "Теперь к стоматологу ходим только так..."

💬 Твой комментарий:
"Слишком много восклицаний. Все посты должны заканчиваться вопросом, не утверждением"

---

*Выводы для следующего поста:*
📌 Глобальное правило добавлено

Правило будет применяться ко всем следующим постам:
"Слишком много восклицаний. Все посты должны заканчиваться вопросом, не утверждением"

Это правило передастся генератору идей (Этап 1) для учета при создании новых вариантов.

[✅ Готово, выкладываю пост] [📋 Модерировать еще]
```


## Токен бота

1. Найти бота в Telegram
2. Сообщить ему `/start`
3. Выбрать действие

Токен должен быть в `~/.claude/api-keys/telegram-bot-token`

## Интеграция с n8n

### Workflow 1: Ежедневная генерация (Пост 1/3)

```
Trigger: Cron (10:00 МСК)
    ↓
Load глобальные_правила.json + модель_рейтинги.json
    ↓
Call скрипт: python idea-generator.py
    (генерирует 5 вариантов, я выбираю лучший)
    ↓
Save to очередь_модерации.json (1 вариант)
    ↓
Send notification to Telegram (ботом):
   "🎯 Пост 1/3 на модерацию!"
   [Лучший вариант от: Claude Sonnet]
   [Активных правил: N]
```

### Workflow 2: После одобрения Поста 1/2/3

```
Trigger: File trigger (generate_next_post.json создан)
    ↓
Пост добавлен в очередь_постов.json
    ↓
IF это был пост 3/3:
    → Done, ждем следующего дня
ELSE:
    → Load глобальные_правила.json
    → Call idea-generator.py для СЛЕДУЮЩЕГО поста
    → Save to очередь_модерации.json
    → Send notification in Telegram:
        "🎯 Пост 2/3 на модерацию!" (или 3/3)
```

## Процесс за день

```
10:00 → Генерация Поста 1
    ↓
Юзер решает (accept/edit/reject)
    ↓
Пост 1 в очередь_постов.json
    ↓
Триггир → Генерация Поста 2
    ↓
Юзер решает
    ↓
Пост 2 в очередь_постов.json
    ↓
Триггир → Генерация Поста 3
    ↓
Юзер решает
    ↓
Пост 3 в очередь_постов.json
    ↓
✅ На день достаточно, три поста готовы к публикации
    ↓
(Юзер сам постит вручную, позже будет автоматический постинг)
```

## Следующий шаг

После модерации пост идет в Этап 5 (Публикация):
1. Медиа подбирается автоматически (из МедиаРотация.json)
2. Пост публикуется по расписанию (из РасписаниеПубликации.json)
3. Метрики собираются (Этап 6)
