# Система соревнования моделей

## Концепция

**5 моделей генерируют 1 вариант каждая → Claude выбирает лучший → Юзер одобряет → Система учит рейтинги**

```
День 1:
5 моделей → 5 вариантов → Claude: "Лучший вариант от Model X"
                                      ↓
                            Бот показывает пользователю
                                      ↓
                            Юзер: "Принял" (или отредактировал)
                                      ↓
                            Model X +1 win (рейтинг растет)
                            Остальные +1 loss

День 2:
Model X вероятность 80% (выше)
Остальные 20% (ниже)
                                      ↓
                    Но каждая модель может быть выбрана!
                    Любая может выиграть завтра
```

## Алгоритм выбора модели

### Weighted Random Selection

Вероятность выбора каждой модели = её текущий win_rate

```python
models = ['Claude', 'Mistral', 'Llama', 'Neural', 'Gemini']
weights = [0.80, 0.53, 0.33, 0.20, 0.50]

selected = random.choices(models, weights=weights, k=5)
# Результат: ['Claude', 'Mistral', 'Gemini', 'Neural', 'Claude']
```

**Что это значит:**
- Claude будет выбран ~4 раза из 5 (вероятность 0.80)
- Mistral ~2.5 раза (вероятность 0.53)
- Llama ~1.5 раза (вероятность 0.33)
- Neural ~1 раз (вероятность 0.20)
- Gemini ~2.5 раза (вероятность 0.50)

### Пересчет рейтингов после одобрения

```
User одобрил вариант от Claude

Claude:
  wins += 1 (теперь 13)
  losses += 0
  rate = 13 / (13 + 3) = 0.81

Mistral:
  wins += 0
  losses += 1 (теперь 8)
  rate = 8 / (8 + 8) = 0.50

... (остальные тоже +1 loss)
```

## Процесс (пошагово)

### 1️⃣ Генерация идеи (Этап 1, idea-generator.py)

```
Cron: 10:00 МСК (ежедневно)
    ↓
Load глобальные_правила.json
    ↓
Select 5 моделей by weighted probability
    ↓
Generate 5 вариантов (по 1 от каждой)
    ↓
Claude analyze & select best
    ↓
Save to очередь_модерации.json
    ↓
Notify user in Telegram: "🎯 Новый вариант готов!"
```

### 2️⃣ Модерация (Этап 5, moderation_bot.py)

```
User: /moderate
    ↓
Bot shows 1 selected variant (from Claude's choice)
    ↓
Buttons:
  [✅ Принять]
  [✏️ Редактировать]
  [⬅️ Назад / Далее ➡️]
  [❌ Отказать всё]
    ↓
User: Принял
    ↓
Save decision with selected_model
    ↓
Update модель_рейтинги.json
    ↓
Move to очередь_постов.json (publishing queue)
```

### 3️⃣ Обновление рейтингов

```python
# В moderation_bot.py при одобрении:

moderator.update_model_ratings(winning_model)

# Функция:
- winning_model.wins += 1
- все остальные.losses += 1
- пересчитать rate для всех
- сохранить в модель_рейтинги.json
```

## Когда менять модели?

- ❌ **Никогда не блокируй** — каждая модель может выиграть
- ✅ **Добавляй новые** — если появилась интересная модель
- ⚠️ **Мониторь** — если модель не выигрывает 2+ недели, может быть проблема

Пример:
```
Если Llama не выиграла за 14 дней → проверить:
- Работает ли Ollama?
- Правильный ли промпт?
- Может ли быть проблема с качеством?
```
