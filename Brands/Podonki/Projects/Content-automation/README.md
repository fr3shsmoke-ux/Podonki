# Podonki Automation Project

Система автоматизации постинга, генерации контента и управления каналами Podonki.

## Структура

```
Podonki/
├── src/                    # Исходный код
│   ├── parsers/           # Парсеры Telegram
│   ├── generators/        # Генераторы контента (AI)
│   ├── schedulers/        # Расписание публикации
│   └── api/               # API интеграции
├── data/
│   ├── datasets/          # Датасеты для обучения
│   ├── raw/               # Сырые данные
│   └── processed/         # Обработанные данные
├── scripts/               # Скрипты для запуска
├── config/                # Конфигурация
├── docs/                  # Документация
└── notebooks/             # Jupyter notebooks для анализа
```

## Функции

- ✅ Парсинг всех каналов (свои + конкуренты)
- 🤖 Генерация постов через Claude/Ollama
- 📅 Автоматический постинг по расписанию
- 📊 Анализ производительности каналов
- 💬 AI в чате для сбора данных
