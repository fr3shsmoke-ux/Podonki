## Как запустить бота

```bash
# 1. Убедиться что токен Telegram сохранён
# ~/.claude/api-keys/telegram-bot-token

# 2. Запустить бота
python scripts/moderation_bot.py

# 3. В Telegram отправить /start
# Появится меню с кнопками

# 4. Нажать "Модерировать посты"
# Увидишь готовый пост про Podonki Classic

# 5. Выбрать действие
# ✅ Принять — пост в очередь публикации
# ✏️ Редактировать — отправить исправленный текст
# ❌ Отказать — отклонить все варианты
```

## Что происходит при нажатии ✅ Принять

1. **Сохраняется решение** в `решения_модерации.json`
   - post_id, decision, variant_text, selected_model, timestamp

2. **Обновляются рейтинги** в `модель_рейтинги.json`
   - Выигравшая модель: +1 win
   - Остальные 4 модели: +1 loss
   - Пересчитывается rate (wins / (wins + losses))

3. **Пост добавляется** в `очередь_постов.json`
   - Готов к публикации в Telegram

4. **Триггирится генерация** следующего поста
   - Создаётся файл `generate_next_post.json`
   - n8n webhook срабатывает (если запущен)
   - Генерируется Post 2/3
