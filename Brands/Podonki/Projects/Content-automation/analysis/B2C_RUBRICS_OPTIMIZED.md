## ИЕРАРХИЯ РУБРИК (по важности для engagement)

### 🏆 ТОП 5 (ОБЯЗАТЕЛЬНЫЕ)

#### 1. **Вкусовой тестер** (вес: 15%)
- Описание вкуса (как в Train Lab: "холодное манго — не приторное")
- Целевая аудитория ("для ценителей", "для экспериментаторов")
- Крепость + тип удара ("50мг, но мягкий")
- Сравнение с другими вкусами своей линейки

**Примеры паттерна**:
```
Чё, пробовал Last Hap [вкус]?
[Описание как впечатление, не как спецификация]
Крепкость 50мг, но [контраст]
Кто любит [категория] — зайдёт. Ссылка в шапке
```

**Эмодзи**: 1 (вкус) или 0

---

#### 2. **Челлендж / Игра** (вес: 12%)
- Угадай вкус по описанию
- Выбери между двумя вкусами (poll-style)
- Вызовы (например, "кто быстрее паримонит")
- Приз = вейп-бокс, мерч, стикеры (НИКОГДА скидку!)

**Примеры паттерна**:
```
Угадай вкус по описанию 🎯
[Атмосфера вкуса, не название]
Пиши в комментариях!
Первые 3 угадавших получат [приз]
```

**Эмодзи**: 2 макс (для игры)

---

#### 3. **Лайфстайл / Момент дня** (вес: 13%)
- Ситуация + вкус (утро/школа/работа/вечер/ночь)
- "Во что ты туда берёшь?" (контекст выбора)
- Настроение + вкус ("меланхолия", "кайф перед выходными")
- Сезонность (зимний вкус, летний)

**Примеры паттерна**:
```
[Ситуация] = Last Hap [вкус]
[Почему именно этот вкус для этого момента]
Ты как, что берёшь в такие моменты?
```

**Эмодзи**: 1

---

#### 4. **Новинка / Анонс** (вес: 11%)
- Новый вкус (первый пост в линейке)
- Новая линейка (когда выходит полностью)
- Возврат популярного вкуса
- Лимитированный выпуск

**Примеры паттерна**:
```
Новинка в линейке Podonki Last Hap - [вкус] 🎉
[Позиция вкуса в линейке]
Уникальность: [что отличает от других вкусов]
```

**Эмодзи**: 1

**ВАЖНО**: НЕ ПИСАТЬ "уникальная формула"! Вместо этого: "Сочетание [ингредиента] + [ингредиента]"

---

#### 5. **Образование / Объяснение** (вес: 10%)
- Как выбрать вкус для своего вейпа
- Чем отличается от других брендов (о Podonki)
- Что такое крепкость, как она влияет на удар
- Где найти свой вкус (по категориям)

**Примеры паттерна**:
```
Как выбрать Last Hap?
Если ты [предпочтение] → берёшь [вкус]
Если ты [предпочтение] → берёшь [вкус]
Все вкусы в шапке
```

**Эмодзи**: 0-1

---

### 🎯 СЕРЕДИНКА (ВАЖНЫЕ, но реже)

#### 6. **Мемы / Тренды** (вес: 8%)
- Вейп-мемы (обычные мемы про вейпинг)
- Тренды из ТикТока/ВК (если релевантны)
- "Типы людей" посты

**Примеры**:
```
Типы парильщиков:
1. "Я за вкус" → Last Hap
2. "Мне нужна мощь" → Critical
3. "Люблю всё сразу" → Sour
```

**Эмодзи**: 0-2

---

#### 7. **Раздачи / Гивэвей** (вес: 7%)
- Вейп-боксы
- Мерч (стикеры, кейсы, худи)
- Лимитированные вкусы
- Условие: репост, подписка, комментарий

**Примеры**:
```
Раздача: вейп-бокс Last Hap 🎁
Что нужно сделать:
- Подпишись на канал
- Репостни пост
- Напиши в комментариях свой любимый вкус
```

**Эмодзи**: 2 (для привлечения)

---

#### 8. **Образование (глубокий разбор)** (вес: 6%)
- "Почему Last Hap лучше конкурентов?" (ОСТОРОЖНО!)
- "Как делают вкусы Podonki?"
- Тестирование: какой вкус более сладкий/кислый

**Примеры**:
```
Сравнили Last Hap Клубника с [конкурент]
Last Hap: сладкая клубника, сливочный финиш
[Конкурент]: кислая, приторная
Last Hap = больше нравится 🥰
```

**ВАЖНО**: Не писать "уникальная формула", не кидать обвинения конкурентов. Только факты.

---

#### 9. **Истории в деталях / За кулисами** (вес: 5%)
- Как создавали этот вкус
- История создания линейки
- Интервью с создателем/тестером

**Примеры**:
```
За кулисами Last Hap
Когда мы создавали Last Hap Манго Айс, нам нужно было:
- Холодный финиш (минт в конце)
- Свежесть манго без приторности
- Мягкий удар (крепкость 50, но не бьёт)
...и вот что получилось
```

**Эмодзи**: 1

---

#### 10. **Вопросы сообщества** (вес: 4%)
- "Какой вкус вы выбираете в такой ситуации?"
- "Какой вкус Last Hap ваш фаворит?"
- Голосование между вкусами

**Примеры**:
```
Между какими вкусами Last Hap ты всегда не можешь выбрать?
1. Клубника Банан
2. Манго Айс
3. Кислый микс
Голосуй в комментариях
```

**Эмодзи**: 1

## МАТРИЦА ВЫБОРА РУБРИКИ

### По дню недели (ротация)
```
Пн: Вкусовой тестер
Вт: Образование / Объяснение
Ср: Челлендж / Игра
Чт: Лайфстайл / Момент дня
Пт: Мемы / Тренды
Сб: Раздача / Гивэвей или Тестирование
Вс: Вопросы сообщества или История
```

### По вкусу (линейка)
```
Last Hap (40 вкусов) → Вкусовой тестер, Лайфстайл
Podgon (25 вкусов) → Образование (бюджет), Мемы
Sour (20 вкусов) → Челлендж (кислота = загадка)
Critical (15 вкусов) → Для крепышей (Образование, Позиционирование)
Hotspot Resonance → Контраст, Настроение
```

## ВАЛИДАЦИЯ ПОСТА

Перед публикацией проверить:

- [ ] Есть ли упоминание Podonki (rule: всегда в заголовке)
- [ ] Названия вкусов строчные ("клубника банан", не "Клубника Банан")
- [ ] Эмодзи: 0-2 максимум
- [ ] Нет скидок/промокодов (только физические призы в B2C)
- [ ] Нет "уникальной формулы", "приобретайте", "ощути"
- [ ] Используется дефис (-), не длинное тире (—)
- [ ] Тон: как совет другу, не реклама
- [ ] Нет восклицательных знаков в конце предложений
- [ ] Кроссинг: этот пост не дублирует вчерашний?
