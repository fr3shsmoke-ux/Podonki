## 🚀 Быстрый старт (3 шага)

### Вариант 1: Google Sheets (для маркетологов)
1. Открой Google Sheets
2. Файл → Импорт → Загрузить файл
3. Выбери `podonki-products.csv`
4. Готово! Используй описания в контент-плане

### Вариант 2: JavaScript (для разработчиков)
```javascript
const products = require('./podonki-products-descriptions.json')
const liquid = products.find(p => p.id === 'liquid_light')
console.log(liquid.detailed_description)
```

### Вариант 3: Python (для аналитиков)
```python
import json
with open('podonki-products-descriptions.json') as f:
    products = json.load(f)
    liquids = [p for p in products if p['type'] == 'liquid']
    print(f"Жидкостей: {len(liquids)}")
```

## ❓ Часто задаваемые вопросы

**Q: Можно ли менять описания?**
A: Да, адаптируй под свой стиль. Главное — сохранить молодёжный тон.

**Q: Где взять названия всех вкусов?**
A: Они в исходных каналах (TrainLab, PodonkiOFF). В следующей версии добавим в базу.

**Q: Как обновлять базу?**
A: Добавляй новые продукты в JSON по образцу существующих.

**Q: Можно ли использовать в других проектах?**
A: Да, это публичные данные. Используй свободно.

**Q: Откуда извлечены данные?**
A: Из публичных Telegram-каналов Podonki (TrainLab, PodonkiOFF, Podonki b2c).
