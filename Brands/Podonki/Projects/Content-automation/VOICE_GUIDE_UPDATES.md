## 1️⃣ ФИРМЕННЫЕ ЭЛЕМЕНТЫ (используй ВСЕГДА)

### Обязательные компоненты

```
🔤 ФИРМЕННОЕ ЛИЦО PODONKI

1. Эмодзи (частота обязательна):
   💀 — символ бренда (используй в подписях)
   🔥 — для горячих предложений / новинок
   🧪 — для ароматизаторов / конструкторов
   👉 — для направления внимания
   🟢 — для списков / маркеров

2. Фирменные фразы (обязательны в 80% постов):
   ✅ "ONLY WHITE PRODUCTS" — главный слоган (легальность)

   ✅ **__Жирный + подчеркивание__** для заголовков
   ✅ ➖ Маркеры для списков
   ✅ Пустые строки между блоками (читаемость)

4. Структура поста:
   [Эмодзи] [Заголовок 1-2 строки]
   [Основной контент, 3-7 строк]
   [CTA или контакт]
   [Фирменная подпись 💀 НАШИ РЕСУРСЫ 💀]
```

## 2️⃣ ТОН И ГОЛОС

### Характеристики (подтверждены анализом)

| Параметр | Рекомендация | Примеры |
|----------|-------------|---------|
| **Формальность** | 🟡 Средний (60% формально, 40% разговорно) | Формально: "Полный комплект документов", "Сертифицированный продукт" |
| | | Разговорно: "Ребят, у нас новая поставка", "Как насчет...?" |
| **Юмор** | ✅ Приветствуется (но не острый) | "Выражение лица твоего друга" как пример кислого, "Отпускается без рецепта врача" |
| **Эмоциональность** | ✅ Позитивная | 😍, 🤩, 🥳, 🔥 — часто |
| **Aggressiveness** | ❌ Не агрессивный | "Быстро закончилась" вместо "СРОЧНО ПОКУПАЙ" |
| **Персональность** | ✅ Дружелюбный, как знакомый | "Ты продавец в VAPESHOP?", "Записывай рецепт" |

### Как НЕ писать

❌ "СРОЧНО! ПОСЛЕДНИЕ ЧАСЫ!" — перепродажа, не Podonki
❌ "Худший продукт на рынке" — сравнение с конкурентами
❌ "Это доказано" — излишние претензии
❌ "Все говорят" — обобщение

### Как писать (правильно)

✅ "Осталось несколько дней" — спокойная срочность
✅ "Только белая продукция" — подчеркивание качества
✅ "Наш клиент Даня сказал..." — социальное доказательство
✅ "Вы спросите..." — диалог с аудиторией

## 3️⃣ ТИПЫ КОНТЕНТА И ШАБЛОНЫ

**Рекомендуемый микс** (на 5 постов):

- 4️⃣ Информация / Юмор (15%)

📅 Сроки: [Дата]

🧑‍💻 Подробнее: @SKULL_manageer

## 4️⃣ ЭМОДЗИ (обязательно!)

### Система эмодзи Podonki

| Эмодзи | Использование | Частота | Примеры |
|--------|---------------|---------|---------|
| **🔥** | Горячие предложения, новинки, розыгрыши | 1-3 на пост | "Розыгрыш 🔥", "новая партия быстро закончилась 🔥" |
| **💀** | Фирменный символ (в подписях) | 1 в конце | "💀 НАШИ РЕСУРСЫ 💀" |
| **🧪** | Ароматизаторы, жидкости, "химия" | 1-2 на пост | "PODONKI 🧪", "кислый ароматизатор 🧪" |
| **👉** | Направление внимания, ссылки | 1-2 на пост | "👉 ссылка", "👉 заказать" |
| **🟢** | Маркеры списков (✅ вместо ➖) | По количеству пунктов | "🟢 Преимущество 1" |
| **👨‍💻👩‍💻** | Контакты менеджеров | 1 на пост (если B2B) | "👨‍💻 Менеджер:", "👩‍💻 Опт:" |
| **😍🤩😁** | Эмоциональные реакции | 1-2 на пост | "😍 Продакт класса", "🤩 Невероятно" |
| **🥳** | Праздничный тон (акции, розыгрыши) | 1-2 на пост | "Стартует в понедельник 🥳" |

### Правило эмодзи
- ✅ 1 эмодзи = 1 смысл (не переусложняй)
- ✅ Порядок эмодзи: сначала действие (🔥), потом символ (💀)
- ✅ Эмодзи в начале заголовка и конца блока
- ❌ Не более 3 разных эмодзи подряд

## 5️⃣ СПЕЦИФИЧНЫЕ СЛОВА И ТЕРМИНЫ

### Словарь Podonki (используй при упоминании товаров)

| Слово | Контекст | Примеры |
|-------|----------|---------|
| **Подгонки** | Название ароматизаторов | "Кислый конструктор Подгонки", "Ароматизатор ПОДОНКИ" |
| **Конструктор** | Pod-система или жидкость для самомиксования | "Кислый конструктор", "классический конструктор" |
| **Белый продукт** | Легальный, сертифицированный, акцизный | "ONLY WHITE PRODUCTS", "Full белый продукт" |
| **ЖТ** | Жевательный табак (сокращение) | "ЖТ Podonki Classic", "ЖТ КЛИК" |

### Как упоминать продукты

❌ "вейп" (слишком широко)
✅ "конструктор", "жидкость", "ароматизатор", "ЖТ"

❌ "нелегальный" (риск)
✅ "ONLY WHITE PRODUCTS", "полный комплект документов"

❌ "самый лучший" (пустое)
✅ "Невероятное качество жидкости", "Стильный дизайн и яркие цветовые палитры"

## 7️⃣ ЧАСТЫЕ ОШИБКИ (избегай)

### Ошибка 1: Забыл фирменную подпись
❌ Пост без "💀 НАШИ РЕСУРСЫ 💀"
✅ ВСЕГДА заканчивай с подписью

### Ошибка 2: Не упомянул "ONLY WHITE PRODUCTS"
❌ "Новый продукт X"
✅ "ONLY WHITE PRODUCTS — новый продукт X"

### Ошибка 3: Слишком формальный тон
❌ "Представляем вашему внимаю юридически сертифицированный..."
✅ "Встречайте пополнение для любителей классических вкусов!"

### Ошибка 4: Нет CTA или контакта
❌ Пост заканчивается информацией
✅ Всегда заканчивай: "Пиши: @SKULL_manageer" или "Заказывай: [ССЫЛКА]"

### Ошибка 5: Слишком много эмодзи
❌ "🔥🔥🔥 Розыгрыш 🎯🎯 Приз 🚗🏆🏆"
✅ "🔥 Розыгрыш 🥳 ... Приз: машина 🚗"

### Ошибка 6: Упомянул конкурентов критически
❌ "Конкурент X делает хуже"
✅ Молча превосходи конкурентов, не упоминай их

## 8️⃣ КОНТРОЛЬНЫЙ ЧЕКЛИСТ ДЛЯ КАЖДОГО ПОСТА

Перед публикацией проверь:

- [ ] **Заголовок** с эмодзи (1-2 строки максимум)
- [ ] **Основной контент** (3-7 строк, 200-400 символов)
- [ ] **Преимущества / Условия** в виде списка (➖ или 🟢)
- [ ] **Явный CTA** ("Жми", "Пиши", "Заказать")
- [ ] **Контакт менеджера** (@SKULL_manageer или @white_podonki)
- [ ] **Фирменная подпись** "💀 НАШИ РЕСУРСЫ 💀"
- [ ] **Минимум 1 фирменный эмодзи** (🔥, 💀, 🧪)
- [ ] **Форматирование** (__жирный__ для важного)
- [ ] **ONLY WHITE PRODUCTS** (если это B2B или продукт)
- [ ] **Проверка опечаток** (нет кракозябр, правильная русская грамматика)

## 🚀 ИСПОЛНЕНИЕ

### Где использовать эту guide
1. 📝 В brief для контент-креаторов
2. 🤖 В промптах для Claude/ChatGPT при генерации контента
3. ✏️ При редактировании существующих постов
4. 📊 Для обучения новых членов команды

### Частота обновления
- Пересматривать **каждый месяц** (после 5-10 новых постов)
- Адаптировать под новые тренды (эмодзи, фразы)
- Сохранять ядро (💀, ONLY WHITE PRODUCTS, фирменные фразы)
