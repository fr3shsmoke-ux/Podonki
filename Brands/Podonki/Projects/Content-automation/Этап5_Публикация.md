# Этап 5 — Публикация в Telegram

**Цель**: автоматически (или по расписанию) отправить готовый пост (текст + картинка + видео) в нужный канал через Telegram Bot API.

## Архитектура

```
Готовые посты (из Этап 4)
    ↓
Очередь публикации (БД или JSON)
    ↓
Telegram Bot API (sendMessage, sendPhoto, sendVideo, sendMediaGroup)
    ↓
Каналы: Podonki, Train Lab, Podonki OFF
    ↓
Логирование (какой пост, когда, ID сообщения)
```

## Telegram Bot API

### Требования

- **Bot Token**: сохранить в `~/.claude/api-keys/telegram-bot-token`
- **Channel ID**: ID каналов (например, `-1001234567890`)
- **Permissions**: бот должен быть админом в каналах

### Методы отправки

#### 1. Только текст
```
POST https://api.telegram.org/bot{TOKEN}/sendMessage
{
  "chat_id": "-1001234567890",
  "text": "Текст поста",
  "parse_mode": "Markdown"
}
```

#### 2. Текст + фото
```
POST https://api.telegram.org/bot{TOKEN}/sendPhoto
{
  "chat_id": "-1001234567890",
  "photo": "https://url/photo.jpg" или FormData с файлом,
  "caption": "Текст с фото",
  "parse_mode": "Markdown"
}
```

#### 3. Текст + видео
```
POST https://api.telegram.org/bot{TOKEN}/sendVideo
{
  "chat_id": "-1001234567890",
  "video": "https://url/video.mp4" или FormData,
  "caption": "Текст с видео",
  "duration": 30,
  "parse_mode": "Markdown"
}
```

#### 4. Медиа группа (картинка + видео)
```
POST https://api.telegram.org/bot{TOKEN}/sendMediaGroup
{
  "chat_id": "-1001234567890",
  "media": [
    {
      "type": "photo",
      "media": "https://url/photo.jpg",
      "caption": "Картинка"
    },
    {
      "type": "video",
      "media": "https://url/video.mp4",
      "caption": "Видео"
    }
  ]
}
```

## Модуль публикации (Python)

### Файл: `scripts/публикация.py`

```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-
"""
Публикация постов в Telegram через Bot API
Поддержка: текст, фото, видео, медиа-группы
Логирование и отслеживание статусов
"""

import json
import requests
import sys
from datetime import datetime
from pathlib import Path
from typing import Dict, List, Optional

if sys.platform == 'win32':
    import io
    sys.stdout = io.TextIOWrapper(sys.stdout.buffer, encoding='utf-8')

class TelegramPublisher:
    def __init__(self, token: str):
        self.token = token
        self.base_url = f"https://api.telegram.org/bot{token}"
        self.channels = {
            "Podonki": -1001234567890,        # заполнить реальным ID
            "Train Lab": -1001234567891,      # заполнить реальным ID
            "Podonki OFF": -1001234567892     # заполнить реальным ID
        }

    def send_message(self, channel: str, text: str) -> Dict:
        """Отправить только текст"""
        chat_id = self.channels.get(channel)
        if not chat_id:
            return {"success": False, "error": f"Unknown channel: {channel}"}

        data = {
            "chat_id": chat_id,
            "text": text,
            "parse_mode": "Markdown"
        }

        response = requests.post(f"{self.base_url}/sendMessage", json=data)
        return self._parse_response(response)

    def send_photo(self, channel: str, photo_path: str, caption: str = "") -> Dict:
        """Отправить фото с подписью"""
        chat_id = self.channels.get(channel)
        if not chat_id:
            return {"success": False, "error": f"Unknown channel: {channel}"}

        files = {"photo": open(photo_path, 'rb')}
        data = {
            "chat_id": chat_id,
            "caption": caption,
            "parse_mode": "Markdown"
        }

        response = requests.post(f"{self.base_url}/sendPhoto", files=files, data=data)
        files["photo"].close()
        return self._parse_response(response)

    def send_video(self, channel: str, video_path: str, caption: str = "") -> Dict:
        """Отправить видео с подписью"""
        chat_id = self.channels.get(channel)
        if not chat_id:
            return {"success": False, "error": f"Unknown channel: {channel}"}

        files = {"video": open(video_path, 'rb')}
        data = {
            "chat_id": chat_id,
            "caption": caption,
            "parse_mode": "Markdown"
        }

        response = requests.post(f"{self.base_url}/sendVideo", files=files, data=data)
        files["video"].close()
        return self._parse_response(response)

    def send_media_group(self, channel: str, media_list: List[Dict]) -> Dict:
        """Отправить медиа-группу (фото + видео)"""
        chat_id = self.channels.get(channel)
        if not chat_id:
            return {"success": False, "error": f"Unknown channel: {channel}"}

        data = {
            "chat_id": chat_id,
            "media": json.dumps(media_list)
        }

        response = requests.post(f"{self.base_url}/sendMediaGroup", data=data)
        return self._parse_response(response)

    def _parse_response(self, response) -> Dict:
        """Парсить ответ от API"""
        try:
            result = response.json()
            if result.get('ok'):
                return {
                    "success": True,
                    "message_id": result['result'].get('message_id'),
                    "date": result['result'].get('date')
                }
            else:
                return {
                    "success": False,
                    "error": result.get('description', 'Unknown error')
                }
        except Exception as e:
            return {
                "success": False,
                "error": str(e)
            }

def load_posts_queue(filepath: str) -> List[Dict]:
    """Загрузить очередь постов"""
    with open(filepath, 'r', encoding='utf-8') as f:
        return json.load(f)

def publish_post(publisher: TelegramPublisher, post: Dict) -> Dict:
    """Опубликовать один пост"""
    channel = post['channel']
    text = post['text']
    media = post.get('media')

    if not media:
        # Только текст
        result = publisher.send_message(channel, text)
    elif media['type'] == 'photo':
        # Фото + текст
        result = publisher.send_photo(channel, media['path'], text)
    elif media['type'] == 'video':
        # Видео + текст
        result = publisher.send_video(channel, media['path'], text)

    # Логирование
    post['status'] = 'sent' if result['success'] else 'failed'
    post['message_id'] = result.get('message_id')
    post['sent_at'] = datetime.now().isoformat()
    post['error'] = result.get('error')

    return post

def main():
    # Загрузить токен
    with open(Path.home() / '.claude' / 'api-keys' / 'telegram-bot-token', 'r') as f:
        token = f.read().strip()

    publisher = TelegramPublisher(token)

    # Загрузить очередь постов
    posts = load_posts_queue('data/очередь_постов.json')

    # Опубликовать
    results = []
    for post in posts:
        if post['status'] == 'pending':
            result = publish_post(publisher, post)
            results.append(result)
            print(f"✅ Published: {post['id']}" if result['status'] == 'sent' else f"❌ Failed: {post['id']}")

    # Сохранить логи
    with open('data/публикация_логи.json', 'a', encoding='utf-8') as f:
        for result in results:
            f.write(json.dumps(result, ensure_ascii=False) + '\n')

    print(f"\n📊 Опубликовано: {len([r for r in results if r['status'] == 'sent'])}/{len(results)}")

if __name__ == '__main__':
    main()
```

## Интеграция с n8n

### Workflow: Публикация по расписанию

```
Trigger: Cron (каждый час в NN:00)
    ↓
Check if post scheduled for this time
    ↓
Load post from очередь_постов.json
    ↓
Call публикация.py via Shell
    ↓
Send notification to admin (Telegram/email)
    ↓
Update очередь_постов.json with status
```

## Возможные сценарии

### Сценарий 1: Мгновенная публикация
```bash
python scripts/публикация.py --post-id post_2026_03_14_001
```

### Сценарий 2: Отложенная публикация (n8n)
1. Пост добавляется в `data/очередь_постов.json` с временем
2. n8n проверяет расписание каждый час
3. Когда время приходит — вызывает публикация.py

### Сценарий 3: A/B тестирование
1. Вариант A публикуется в основной канал
2. Вариант B публикуется в тестовый канал (или отложенно)
3. Через 24 часа анализируем ER обеих вариантов
4. Сохраняем результаты в `data/AB_тесты.json`

## Ошибки и обработка

| Ошибка | Причина | Решение |
|--------|---------|---------|
| `Invalid chat_id` | Неправильный ID канала | Проверить ID в `channels` |
| `Not enough rights` | Бот не админ в канале | Добавить бота админом |
| `Message too long` | Текст > 4096 символов | Сократить текст |
| `File not found` | Файл медиа не существует | Проверить путь |
| `Connection timeout` | Проблема с интернетом | Повторить через 30 сек |
