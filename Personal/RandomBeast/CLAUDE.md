# CLAUDE.md — RandomBeast

## Проект
Клон/редизайн сайта RandomBeast. Веб-приложение с визуальным контентом.

## Структура
```
original-randombeast/        — оригинальный сайт (референс)
original-design-analysis/    — анализ дизайна оригинала
randombeast-clone/           — основной клон
randombeast-design-clone/    — дизайн-клон
randombeast-screenshots/     — скриншоты для сравнения
clone-comparison/            — сравнение оригинал vs клон
browser-demo/                — демо в браузере
live-demo/                   — живая демо-версия
live-randombeast-capture/    — захват live-версии
vision-control/              — контроль визуала
```

## Стек
- HTML/CSS/JavaScript
- Playwright (для захвата и тестирования)

## Команды
```bash
# Открыть демо в браузере
start browser-demo/index.html

# Сравнение с оригиналом
# Используй скриншоты из randombeast-screenshots/

# Деплой
/deploy github
```

## Правила
- Сохранять визуальное соответствие оригиналу
- Каждое изменение дизайна — сначала скриншот до/после
- Не удалять папки с анализом и сравнениями — это референс
- Адаптивность: мобилка, планшет, десктоп
