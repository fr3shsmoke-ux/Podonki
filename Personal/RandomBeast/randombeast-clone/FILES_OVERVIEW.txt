=====================================================
ОСНОВНОЙ ФУНКЦИОНАЛ
=====================================================

✅ БЭКЕНД:
- Telegram Web App инициализация с верификацией (HMAC-SHA256)
- CRUD операции для розыгрышей (создание, редактирование, удаление)
- Система участников (присоединение, отслеживание)
- Случайный выбор победителя
- Система уведомлений (placeholder)
- Админ панель с модерацией
- SQLite БД с 4 таблицами (users, giveaways, participants, notifications)

✅ ФРОНТЕНД:
- Responsive дизайн для мобильных
- 3 основные вкладки (Active / Participations / Created)
- Модальное окно для создания розыгрышей
- Real-time взаимодействие с API
- Интеграция с Telegram Web App API
- Красивое оформление (градиент, анимации)

✅ ДЕПЛОЙ:
- Docker & Docker Compose готовые конфиги
- Поддержка .env переменных
- Health check для мониторинга

✅ АВТОМАТИЗАЦИЯ:
- Полный гайд для интеграции с n8n
- Примеры workflows (auto-create, daily-end, notify-winner)
- Webhook endpoints для n8n

=====================================================
КАК НАЧАТЬ
=====================================================

1. БЫСТРЫЙ СТАРТ (5 минут):
   - Прочитай QUICKSTART.md
   - Получи Bot Token от @BotFather
   - Создай .env файл
   - npm install && npm run dev

2. ПОЛНЫЙ ГАЙД:
   - Прочитай SETUP.md
   - Следуй всем шагам
   - Протестируй через curl/Postman
   - Интегрируй с n8n (docs/N8N_INTEGRATION.md)

3. ДЕПЛОЙ:
   - Используй docker-compose или вручную на VPS
   - Обнови Bot Token в .env
   - Запусти приложение

=====================================================
API ENDPOINTS (всего 11)
=====================================================

TELEGRAM:
  POST /api/telegram/init
  GET /api/telegram/user/:telegramId

GIVEAWAYS:
  POST /api/giveaway/create
  GET /api/giveaway/active
  GET /api/giveaway/:giveawayId
  POST /api/giveaway/:giveawayId/join
  POST /api/giveaway/:giveawayId/end

USER:
  GET /api/user/:userId/giveaways
  GET /api/user/:userId/participations
  GET /api/user/:userId/notifications

ADMIN:
  GET /api/admin/stats
  GET /api/admin/giveaways
  DELETE /api/admin/giveaways/:giveawayId
  POST /api/admin/users/:userId/ban
